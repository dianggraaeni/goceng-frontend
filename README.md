# goceng-frontend
GOCENG Frontend is a web interface for GOCENG, a financial tracking and monitoring platform integrated with a WhatsApp chatbot. The website helps users track income, expenses, view financial reports, and get insights through a modern, interactive, responsive interface with bilingual support (Indonesian and English).

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
